End to End Data Analytics Project (Python + SQL+ Tableau)

Complete End-to-End Analysis of Airbnb Listings in New York City: Data Cleaning with Python, Data Analysis with MySQL & Visualization with Tableau
In this project, a comprehensive end-to-end analysis was conducted on the New York City Airbnb Open Dataset sourced from Kaggle.

Part 1:
Python was utilized to:
  • Download the dataset from Kaggle
  • Clean and preprocess the data
  • Load the processed data into MySQL for analysis
Part 2:
MySQL was utilized to perform an in-depth analysis of the Airbnb dataset, covering:
  • Basic Exploration
  • Host Insights
  • Price Trends
  • Customer Engagement & Ratings
  • Geographic Distribution
  • Time-Based Analysis
  • Data Quality Assessment
Part 3:
Tableau was used to create an interactive dashboard that presents critical insights, such as:
  • Geographical Analysis
  • Monthly Bookings Segmented by Neighborhood Group
  • Total Count of Neighborhoods within Each Neighborhood Group
  • Top 5 Host IDs with the Highest Number of Listings per Neighborhood Group
  • Top 10 Host IDs by Number of Listings
  • Distribution of Room Types
  • Bookings Totals by Neighborhood Group and Room Type
  • Annual Review Totals
  • Top 10 Neighborhoods Ranked by Average Price
  • Average Monthly Reviews by Room Type & Neighborhood Group
  • Average Price Broken Down by Neighborhood Group & Room Type
